## My Blog

### monorepo

this is a monorepo base on yarn([yarn berry](https://github.com/yarnpkg/berry)) workspaces.

- packages/client: frontend. built with `react`, `tailwindcss`, `vite`.
- packages/server: backend. built with `koa`, `mongoose`.
